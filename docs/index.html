shadowcloud

shadowcloud creates isolated filesystem in your cloud, which provides transparent encryption and enhances your clouds with many additional features

Strong encryption

AES-256, ChaCha20 and Elliptic Curve cryptography

Zero knowledge

Your keys stored exclusively on your devices

Open-source

Free and contains no backdoors, crossplatform

Saves space

Uses direct network streams, without the need for a local drive usage

Enhances your cloud

No file size limits, unlimited versioning, mount as local drive, etc

Smart metadata

Extracts metadata, creates text and image previews for your files

Virtual Data Regions

Use any amount of cloud storage accounts as a united data partition, easily split or replicate your data between multiple cloud storages.

Leading Encryption Technology

ShadowCloud provides on-the-fly encryption and integrity checking. Just copy your key to other device and start using your files.

Immutable Filesystem

Once written, your data will never be lost. Use history to restore previous state not only for files, but for entire directory tree.
